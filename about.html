Why migrating to DevOps?

Since I decided to take my first Cloud certification, I fell in love with cloud and its subjects.

DevOps is a culture that helps teams to work better together, and it's a way to improve the development process.

This led me to learn more about DevOps and its practices like CI/CD, and tools that make this happen like Terraform, Git, GitHub Actions, Docker, Kubernetes, and so on.

I like to code, but I'm not a developer. So, I'm learning to code in my spare time and using my experience in IT infrastructure to build and automate things, which allows me to code something I enjoy and find useful.
